Arbetsprocess

Jag har använt nodeJS med parcel för att automatisera uppgifter vid utveckling och publicering av en webbplats. Jag har följt instruktionerna under Teori och läsanvisningar samt från föreläsningarna för att göra uppgiften. Jag har använt kommandot npm run build för att minifiera min kod och inställningen med browserlists i package.json för att skapa bakåtkompalitet i webbläsare för JavaScript. Jag har använt picture-elementet i HTML med följande format på bilder: "image.jpg?as=webp&width=1024" för att optimera dem automatiskt. Jag har använt kommandot npm run build för att publicera webbplatsen.

Anna Danielsson, Webbutvecklingsprogrammet
