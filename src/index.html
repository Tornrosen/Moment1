Beskrivning av uppgiften

Denna uppgift går ut på att börja använda npm och parcel för att automatisera uppgifter vid utveckling och publicering av en webbplats. Dessa uppgifter är att minifiera CSS, JavaScript och HTML, transpilera JavaScript, optimera bilder, skapa en bakåtkompabilitet i filerna så att JavaScript transpileras i äldre webbläsare samt att kopiera arbetsfiler till en mapp för publicering med optimerade filer.

Anna Danielsson, Webbutvecklingsprogrammet
